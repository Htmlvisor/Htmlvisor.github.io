Editor HTML

Instrucciones de uso:

Escribe código HTML en el área de arriba. La vista previa se actualizará automáticamente en tiempo real.

Ejemplo:

<h1>Mi página web</h1>

Para incluir imágenes, usa la etiqueta <img src="url">.
